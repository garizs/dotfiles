# Global agent guidance

## Codebase search
- If `grepai` is available and the task requires finding existing patterns inside a codebase, use it before manual file-by-file exploration.
- Prefer existing repository patterns over inventing new architecture when an equivalent pattern already exists.

## Clarification workflow
- Prefer the `interview` tool over many follow-up chat questions when there are 3 or more clarifications, multiple options/tradeoffs, or when screenshots/images/diffs would help.
- Use plain chat for tiny one-off clarifications.
